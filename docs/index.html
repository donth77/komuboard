Komuboard

A free-to-run, realtime, online collaborative whiteboard that works everywhere — desktop, mobile, and VR. Open a link, land in a room, draw together. No signup.

  • Infinite pan/zoom canvas — freehand pen, sticky notes, shapes, text, with live multiplayer sync.
  • Cross-reality, one shared document — the same Yjs board renders in 2D and immersive VR.
  • Anonymous rooms by URL — realtime labeled cursors and presence; survives reconnect on edge persistence.
  • $0 to run — Cloudflare Pages + Workers + Durable Objects at meaningful concurrency.
